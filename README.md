# Online-Quiz-Application-Java
A console-based Online Quiz Application built using Core Java, following Object-Oriented Programming, Collections Framework, and timer-based question handling. This project demonstrates strong Java fundamentals with clean logic, data structures, and interactive user experience.
## Features
- MCQ-based quiz  
- Timer: 10 seconds for each question  
- Automatic score calculation  
- Leaderboard using `TreeMap`  
- Questions stored using `ArrayList`  
- Clean OOP structure  
- Fully console-based — no external libraries required  

## Tech Stack
- Java (Core Java)
- OOP Concepts  
- Collections Framework  
  - ArrayList
  - TreeMap (descending order leaderboard)
- Exception Handling  
- Timer using `System.currentTimeMillis()`
# Project Structure
OnlineQuizApp.java
│── Question class
│── Quiz logic (timer + score)
│── Leaderboard using TreeMap
